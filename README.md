Финальное тестовое задание QAP76 (PJ-04)
Aвтоматизированные тесты с использованием PyTest и Selenium для
интернет-магазина https://www.onlinetrade.ru/

# Требования:
1) Установленный в одну из папок переменной среды PATH операционной системы ChromeDriver
(В консоли Windows можно посмотреть значение этой переменной, выполнив команду echo %TEMP%, в консоли PowerShell необходимо для этого выполнить команду echo $Env:TEMP, а в консоли Linux или MacOS – команду echo $TEMP.)

ChromeDriver - WebDriver for Chrome:
	- https://chromedriver.chromium.org/home
	- Папка /distrib в проекте

# test_basket.py - тесты функциональностей корзины
	- тест1: Проверка изменения счетчика товаров в корзине - при добавлении одного товара счетчик меняется
	- тест2: Проверка счетчика товаров в корзине - при добавлении трех товаров в корзину счетчик равен 3
	- тест3: Добавление товара в корзину - Найти первый товар -> нажать "Купить" -> перейти в "Корзину" -> Убедиться что есть товар, получив его атрибут "Код товара: ХХХХХ"
  	- тест4: У добавленного товара в корзину проверить, что сумма = сумме заказа ИТОГО
	- тест5: Увеличение количества товара в корзине - Найти первый товар -> нажать "Купить" -> перейти в "Корзину" -> Нажать "+" Убедиться кол-во товара увеличилось
    	- тест6: Уменьшение количества товара в корзине - Найти первый товар -> нажать "Купить" -> перейти в "Корзину" -> Нажать "+" -> Нажать "-" Убедиться кол-во товара уменьшилось
	- тест7: Уменьшение количества товара в корзине менее 1 - Найти первый товар -> нажать "Купить" -> перейти в "Корзину" -> Нажать "-" Убедиться, что кол-во товара нельзя уменьшить менее 1  
 	- тест8: У добавленного товара в корзину проверить, что меняется сумма при увеличении кол-ва
  	- тест9: У добавленного товара в корзину проверить, что меняется сумма при уменьшении кол-ва
 	- тест10: У добавленного товара в корзину проверить, что меняется сумма заказа ИТОГО при увеличении кол-ва
  	- тест11: У добавленного товара в корзину проверить, что меняется сумма заказа ИТОГО при уменьшении кол-ва
  	- тест12: Проверка очистки корзины: Найти первый товар -> нажать "Купить" -> перейти в "Корзину" -> Нажать "Очистить корзину" -> Нажать "подтвердить очистку" - Убедиться, что кол-во товара в корзине = 0
 	- тест13: У добавленного товара в корзину проверить, что при изменении кол-ва сумма меняется кратно
